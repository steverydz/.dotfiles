# Dotfiles

## MacOS

```
brew install stow
```

```
cd ~/.dotfiles
stow git
stow nvim
stow tmux
stow vim
stow zsh
```

```
cd ~/.config/nvm
vim .
:MasonInstall black
:MasonInstall emmet_language_server
:MasonInstall flake8
:MasonInstall lua_ls
:MasonInstall prettier
:MasonInstall pyright
:MasonInstall stylua
:MasonInstall ts_ls
```
